O que é

Git é um sistema distribuído de controle de versão livre e de código aberto projetado para lidar pequenos e grandes projetos com velocidade e eficiência.

Vantagens

  • Branching and Merging
  • Small and Fast
  • Distributed
  • Data Assurance
  • Staging Area
  • Free and Open Source

Comandos


git init                         | Cria um novo repositório local
----------------------------------------------------------------------------------------------------
git clone                        | Cria uma cópia de um repositório local ou remoto
----------------------------------------------------------------------------------------------------
git add[arquivo]                 | Add um arquivo
git add *     	                | Add vários arquivos
----------------------------------------------------------------------------------------------------
git commit                       | Envia modificações para o cabeçalho
----------------------------------------------------------------------------------------------------
git push                         | Envia  modificações para o branch master o repositório remoto
----------------------------------------------------------------------------------------------------
git status                       | Lista as modificações e os arquivos que foram criados
----------------------------------------------------------------------------------------------------
git remote add origin[server]    | Conecta a um repositório remoto
